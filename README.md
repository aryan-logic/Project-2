# Project-2
🎓 Student Performance Analyzer

This is a Python-based project that helps manage and analyze student academic performance using pandas and NumPy.

The program allows users to store student marks, view data, and perform meaningful analysis such as identifying toppers, calculating averages, and evaluating overall class performance.

🚀 Features
➕ Add student details and marks
📂 Store data in CSV format
👀 View all student records
📊 Analyze performance:
Average marks per student
Topper of the class
Lowest scoring student
Overall class average


🛠️ Technologies Used
Python
pandas
NumPy

📌 How It Works
User selects options from a menu
Data is stored in a CSV file
pandas is used for data handling
NumPy is used for numerical analysis
